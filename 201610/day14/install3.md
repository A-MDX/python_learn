# 安装第三方模块

在python中，安装第三方模块，是通过包管理工具`pip`完成的。

一般 mac | linux 都已经安装完成。

windows按照pythn时应该装好了。
安装第三方库，像是这样，
首先要知道名字：

```
C:\Users\A-mdx\PycharmProjects\python_learn>pip install Pillow
```
